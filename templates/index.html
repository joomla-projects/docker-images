Help test Joomla!

Warning! These packages are for testing purposes only and should never be used on production websites.

These pull request packages are snapshots of specific code changes including new features and bug fixes.

They are made available to make it easier for users to:

  • Test their websites and extensions for potential issues with a specific pull request.
  • Confirm that the pull request request works as described.
  • Provide feedback on ways to improve it before being accepted.

As these packages are snapshots of a single pull request you may encounter other issues compared to the stable release.

If you do encounter an issue, please check our issue tracker to see if it has already been reported. If a report hasn't been made, please report it so other members of the community can review and fix the issue.
